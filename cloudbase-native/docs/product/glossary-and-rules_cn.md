# 术语与规则

## 主要术语

### Home

展示今日焦点和每日挑战。

### Mission

当前用户正在执行的任务。

### Collab

当前用户创建、且仍需关注的任务。

### Archive

完成或待检视快照记录。

### Closed

任务被关闭，但不等于归档完成。

### Offline Reward Promise

发起者承诺的线下奖励文本，例如请喝奶茶、请吃饭。

## 关键规则

- 刷新依赖进入页面、下拉、操作后刷新与冲突刷新
- 自建自接任务要特别注意 creator / assignee 双重身份
- `review_pending` 阶段只能完成或退回继续
- overdue 与 due today 的任务要优先进入 Today Focus
