# 集成检查清单

## 目标

用于在真实 CloudBase 环境中集中验证一遍主流程。

## 环境准备

1. 在微信开发者工具中打开 `cloudbase-native`。
2. 确认 `miniprogram/config/cloud.js` 指向目标环境。
3. 创建集合：`users`、`tasks`、`task_archives`。
4. 配好订阅模板与 AI 相关环境变量。
5. 上传四个生产云函数。
6. 为 `subscribeScheduler` 配置定时触发器。

## 建议验证顺序

1. 启动并自动建档。
2. 修改昵称并确认持久化。
3. 创建普通任务。
4. 创建自接任务。
5. 同时修改多个子任务进度。
6. 提交检视或直接完成。
7. 查看 archive。
8. 从一个账号分享协作任务给另一个账号。
9. 通过分享链接直接打开并接取。
10. 验证退回继续、完成、重构相关流程。
11. 人为制造一次 stale write，确认触发冲突刷新。
12. 验证订阅授权与定时提醒。
